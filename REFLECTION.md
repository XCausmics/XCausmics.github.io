1. the main difference between flex-direction: row and flex-direction: column is that when row is used, items are placed horizontally from left to right. While if column is used the items would be arranged from top to bottom.  row is mainly used on pc while column is used on mobile devices due to the lack of space. It is noted that row is the default, hence why we don't declare it in the container.
2. It is important to use %, bh, vw, and rem because it allows to change according the the ratio of the screensize instead of a static value. These units would scale based on the screen size instead making it elements properly sized and maintain readability.
